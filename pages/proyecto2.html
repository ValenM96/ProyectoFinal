Analog Altimeter

The Analog Altimeter project was an opportunity to push my hard surface and boolean modeling skills to the maximum. With this work, I set out to create a detailed render of an analog altimeter, exploring advanced modeling techniques to achieve a realistic and functional appearance. This project challenged me to master modeling and rendering tools to produce a high-quality final result.

Modeling: Blender
Texturing: Blender
Rendering: Blender (Cycles)
Composition: Photoshop
